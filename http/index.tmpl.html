This is a generator for the First Ones language from the She-Ra and the Princesses of Power series.

You type words in the box above, press the ‚ú® button, and you'll see them written in the First Ones language.

English and Spanish are supported.

Use "/" to split a word on the word line (for style purposes), and spaces to separate between words.
You can force a language by prefixing a word with the language code. This also allows mixing languages in the same input.

For other languages, or manual input, you can write the glyphs directly using the names from the official First Ones Language definition.
